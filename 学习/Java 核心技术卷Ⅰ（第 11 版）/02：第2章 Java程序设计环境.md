## 2.2 使用命令行工具

- shebang：在 JDK 11 中，单个源文件不再需要 javac 命令。这个特性是为了支持以 “shebang”（#!）行（`#!/path/to/java`）开头的 shell 脚本。

> **什么是 shebang：** 长期以来，shebang 都没有正式的中文名称。Linux 中国翻译组的 GOLinux 将其翻译为“释伴”，即“解释伴随行”的简称，同时又是 shebang 的音译。
